## 1. 概述

门派是玩家在游戏中加入的核心组织，是探索、成长和挑战的主线载体。

- **首发开放 6 个初始门派**：少林寺、玄一教、五岳剑宗、十二峒、唐门、刀宗。
- **加入限制**：少林寺**只招男性**，其余门派不限性别。
- **赛季制关联**：门派开放、势力、副本随赛季更新（标准 3 个自然月一赛季），详见 [00-总览](00-总览.md)。

> 门派基础一览见 [`data/门派一览.csv`](../data/门派一览.csv)。

## 2. 通用机制（师门系统）

所有门派共享同一套拜师、晋阶、判门、学艺规则，差异只在门派任务内容与武学。

### 2.1 弟子阶位

| 阶位 | 师门任务上限/日 | 任务侧重 |
|------|:---------------:|----------|
| 入门弟子 | 20 个/日 | 送货、打杂 |
| 一阶弟子 | 10 个/日 | 采买材料、PVE 野怪 |
| 二阶弟子 | 5 个/日 | 少量 PVP 任务 |
| 三阶弟子 | 2 个/日 | 以 PVP 任务为主 |

> 晋阶顺序固定：入门 → 一阶 → 二阶 → 三阶；满足条件（门派好感度、贡献度）即可晋升。

### 2.2 拜师

- 现有 6 个门派，满足条件即可拜师，拜师顺序非固定。
- 入门即增加对应门派**好感度 +100**。
- 性别限制：少林只招男性。

### 2.3 判门

| 方式 | 触发 | 后果 |
|------|------|------|
| 主动判门 | 玩家向当前师父提出请求，需满足一定条件 | 脱离门派 |
| 自动判门 | 与所在门派好感度降为负数 | 自动判门，且 48 小时内有几率遭门派暗杀（清理门户） |

### 2.4 学习武学

- 条件：满足门派**好感度 + 贡献度**要求。
- 代价：花费潜能等资源向当前师父学习。
- 约束：学到的武学**等级不超过当前师父**。

### 2.5 师门任务设计原则

- 全手动完成（非挂机）。
- 任务种类多样，各门派内容不同，不同阶位获取的任务不同。
- 奖励定位：中量潜能、中量经验、材料 / 银两来源、少量黄金（具体数值见后续《任务系统》）。

## 3. 初始门派

> 每个门派结构统一：起源 → 武器 → 武学体系 → 特色 → 师门任务。武学的**具体招式数值与技能树**留待《武学系统》补全，此处只列名称与定位。

### 3.1 少林寺

- **定位**：禅武合一，内外兼修、刚柔相济；武林泰山北斗。**只招男性。**
- **主要武器**：棍、杖、拳套。
- **代表武学**：少林长拳（基础刚猛拳法）、少林棍法（招牌）、达摩易筋经（内功巅峰）、少林七十二绝技（绝技总称）。
- **特色**：禅武合一，修武同时研读佛经、领悟禅机。

| 师门任务 | 内容 |
|----------|------|
| 宣扬佛法 | 去指定寺庙宣扬佛法，有几率触发战斗，胜利或宣扬完毕后回禀师父 |
| 挑水 | 领水桶到水井打两桶水（一次一桶）交给烧火僧 |
| 敲钟 | 钟楼敲钟三次，全少林可闻；为听到钟声的少林玩家加 30 分钟防御 +2% buff |
| 砍柴 | 松树林砍木→木墩劈柴→交烧火僧 |
| 炼药 | 药王殿找明斛领材料，丹炉炼活血散后交付 |
| 抓捕叛僧 | 指定地图击败叛僧交戒律院；有几率出现放人选项（得银两+神秘道具，但任务失败） |
| 门派挑战 | 出门击败挑衅者 |

### 3.2 玄一教

- **定位**：道门剑修，阴阳五行、天人合一；剑道之宗，道法之源。
- **主要武器**：玄一剑（标志）、太极双剑（衍生，一阴一阳）。
- **代表武学**：玄一剑法（基础，阴阳相生）、五行剑诀、乾坤御剑术（御剑远攻）、道心通明（修行境界）。
- **特色**：内外兼修、身心合一，以剑会友、以道论心。

| 师门任务 | 内容 |
|----------|------|
| 炼药 | 炼丹房李凌霄领材料→炼丹炉炼丹→交付 |
| 送药 | 李凌霄处领药包，限时送达后返回提交 |
| 救援 | 寻找遇困同门，战斗胜利后回师门交差 |
| 巡检 | 指定范围走动，随机触发与窃贼战斗，胜利交差 |
| 杂事 | 打扫演武坪 / 擦拭三清殿祖师像 / 整理藏经阁 / 给药草园施肥 |
| 挑战 | 解剑岩教训闹事者，限时十分钟，打赢交差 |
| 修缮 | 库房领材料，到指定地点（廊桥 / 闻道亭 / 五行宫）修整 |
| 指导（备选） | 演阵台与弟子切磋，获胜交差 |

### 3.3 五岳剑宗

- **定位**：以剑为主、快慢相间的正道剑派；剑乃心之延伸。
- **武学体系**：快剑（以快制敌）、慢剑（以静制动、后发先至）、剑意合一（快慢融合的最终境界）。
- **架构**：掌门负责制 + 长老会监督制，下设五大剑院对应五岳之峰。
- **特色**：山川灵气熏陶，兼修诗词琴棋书画等传统文化。

| 师门任务 | 内容 |
|----------|------|
| 酿酒原料（备选） | 鸣弦泉打一桶水酿酒（骑马会洒水） |
| 清理叛徒（备选） | 按指示地点击杀叛徒 |
| 维持安宁（备选） | 山脚松林解决伤人猛虎 |

> 五岳剑宗师门任务老案多为备选（`//` 标注），后续需补全正式任务列表。

### 3.4 十二峒

- **定位**：苗疆用毒一脉，医毒同源、毒术与医术并重。
- **主要武器**：毒鞭、毒囊。
- **代表武学**：万毒归宗（至高，剧毒领域）、幽冥毒步（身法）、蛊术（培育控制蛊虫）。
- **特色**：以用毒为主但重自然和谐，弟子兼修医术、药理、生态。
- **NPC**：冷婵（入门师父）、碧瑶（一阶师父）、于自谷（二阶师父）。

> 师门任务待补（老案此节只列了 NPC，未写任务）。

### 3.5 唐门

- **定位**：巴蜀暗器世家，机关 + 毒术，以最小之力制敌于无形。**世袭传承，要职唐姓。**
- **武学体系**：爆裂镖（击中爆炸）、毒镖（麻痹 / 剧毒）、轻功身法、暗器制作、毒术研究（毒经阁）。
- **架构**：掌门 + 四大堂口（暗器制作 / 武学传授 / 情报收集 / 医毒研究）。
- **门规**：避世不争、以武止戈。

| 师门任务 | 内容 |
|----------|------|
| 终止谣言 | 唐家集击杀散布谣言者，回禀师父 |
| 排除危险（备选） | 击杀潜入绝情竹海的窥探者并获取目的 |
| 寻回物品（备选） | 找回被盗的绝情散炼制图谱 |
| 炼制毒药（备选） | 药田采腐心草 + 唐景仁处领寒蜈蜕 → 炼断肠散交师父 |
| 打理药田（备选） | 唐云逸处接任务，按要求浇灌药丛 |

### 3.6 刀宗

- **定位**：红枫镇刀法一脉，性格豪迈、快意恩仇；高暴击、命中偏低。
- **武学体系**：狂风快刀（速度）、霸王斩（力沉）、刀宗三式（**累积刀意**，越战越强分层 buff）、影踪刀法。
- **架构**：宗主 + 长老 + 堂主。
- **特色**：以刀法论高下、以实力定地位；内外兼修增强刀法稳定性。

| 师门任务 | 内容 |
|----------|------|
| 维持安宁 | 藏锋集劝说 / 教训闹事小混混 |
| 清理叛徒 | 按指示地点击杀刀宗叛徒 |
| 运送铁胚 | 铁火堂刀破穹领 8 个玄铁铁胚送各地打铁铺 |
| 救援同门 | 解救被劫持的运矿弟子，回师门报告 |
| 查明情况 | 红枫谷探查可疑江湖客，击败取情报拼凑真相 |
| 挖矿 | （待补） |
| 帮助总管 | 大摆筵席数酒 / 闻香院接樱花酿酒（读条到"马上落地"时伸手） |
| 读书答题 | 凌霄阁读书 → 回总管处答题 |
| 同门过招 | 与同门切磋（核心弟子需与凌霄阁顶层两人切磋） |
| 击败挑战者 | 门外击败其他门派挑战者 |
| 门派巡逻 | 巡逻十个地点找小偷，概率遇到打一架 |
