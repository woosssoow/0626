# 03 · 武学系统

> **本文只定义武学的通用规则**，不列任何具体武学。各门派的武学招式、名称与数值留待后续《武学列表》填充。
>
> 关联文档：四维属性见 [01-角色系统](01-角色系统.md)；门派职能与学习/秘籍/三上限的门派侧规则见 [02-门派系统](02-门派系统.md) §2.4。

## 1. 概述

武学是角色战斗能力的核心载体。本系统规定武学**如何分类、如何成长、如何学习、如何受限**，不规定"有哪些武学"。

- 武学的实际效果由角色**四维属性**（力道 / 气劲 / 根骨 / 身法）驱动结算，见《角色系统》。
- 门派的职能定位（坦克 / 输出 / 治疗 / 增益）由其**武学体系**决定，见《门派系统》。
- 武学在战斗中的释放、内力消耗、冷却、连招等**战斗侧规则**归《战斗系统》，本文只到接口为止。

## 2. 武学构成与分类

### 2.1 按功能分类

一门武学按其作用归入下列类别之一（同一门派的武学体系通常覆盖多类）：

| 类别 | 作用 | 主被动 |
|------|------|--------|
| 内功心法 | 提升内力 / 气血 / 抗性等基础属性 | 以被动为主 |
| 外功招式 | 主动攻击、造成伤害 | 主动 |
| 身法轻功 | 移动、闪避、位移、机动 | 主/被动 |
| 辅助武学 | 治疗、增益（buff）、减益（debuff）、控制、驱散 | 主动 |
| 护体绝技 | 减伤、护盾、抗控等保命手段 | 主/被动 |

### 2.2 按战斗作用

战斗作用标签用于和门派职能对齐，一门武学可带多个标签：

`攻击` · `防御` · `治疗` · `增益` · `减益` · `控制` · `位移` · `续航`

> 例：坦克门派以「防御 / 控制 / 护体」为主，治疗门派以「治疗 / 续航 / 驱散」为主——具体由各门派武学体系填充。

## 3. 武学品阶

武学按强度分四阶，品阶决定**强度区间、解锁方式、是否需秘籍、等级上限范围**。

| 品阶 | 定位 | 解锁方式 | 是否需秘籍 |
|------|------|----------|:----------:|
| 基础武学 | 入门基本功 | 潜能直接学 | 否 |
| 进阶武学 | 成长期主力 | 潜能直接学 | 否 |
| 上乘武学 | 高级武学 | 贡献购秘籍解锁上限 + 潜能学 | ✅ |
| 镇派武学 | 门派最强 | 贡献购秘籍解锁上限 + 潜能学（要求更高） | ✅ |

> 品阶越高，等级上限越高、效果越强、解锁门槛越高。各阶的具体等级区间属于数值，待《武学列表》定。

## 4. 武学等级与成长

- 每门武学拥有**独立等级 Lv**，从 1 起，互不影响。
- **升级消耗潜能**（通用成长资源）；高级武学还需先满足解锁上限（见 §5）。
- 等级提升带来该武学**效果数值的提升**（伤害 / 治疗量 / 增益幅度 / 冷却等，具体随武学而定）。
- 武学等级**不会超过其当前可学上限**，上限由三种来源共同约束（见 §5）。

> 升级所需潜能曲线、各阶等级区间等**数值**留待《武学列表》，本文不定。

## 5. 学习与等级上限（三上限）

学习规则与门派强绑定，门派侧完整流程见 [02-门派系统 §2.4](02-门派系统.md)。本节从**武学等级**视角收口核心规则。

### 5.1 两种学习方式

| 方式 | 适用品阶 | 流程 |
|------|----------|------|
| 潜能直学 | 基础 / 进阶 | 花潜能向当前师父直接学 |
| 秘籍解锁 + 潜能学 | 上乘 / 镇派 | 先花门派贡献购秘籍（上 / 中 / 下三册）解锁上限，再花潜能向师父学 |

### 5.2 三种等级上限

一门武学的可学等级同时受三个上限约束，**实际可学等级取其中最低值**：

| 上限 | 来源 | 作用 |
|------|------|------|
| 秘籍解锁上限 | 贡献购上 / 中 / 下三册逐段解锁 | 理论天花板，未购则练不上去 |
| 师父可教上限 | 师父的「可教等级」属性（通常低于师父自身等级） | 师父能带到的上限 |
| 自修可达上限 | 读书 / 实战顿悟等自修途径 | 脱离师父后的突破途径 |

> **当前可学等级 = min( 秘籍解锁上限 , max( 师父可教上限 , 自修可达上限 ) )**
>
> 秘籍是天花板；天花板之下，「师父教」与「自己修」是两条并行抬高途径，取较高者。

## 6. 秘籍与书籍

- 仅**上乘 / 镇派武学**需要秘籍解锁等级上限；基础 / 进阶武学无需。
- 秘籍一般分 **上、中、下三册**，用**门派贡献**购买。
- 每册解锁该武学的**一段等级上限**（上 → 中 → 下，逐段提升）。
- 贡献来源：门派每日任务及其他渠道（势力、活动等）。

## 7. 自修途径

当玩家对某武学的可学需求**超过师父可教上限**后，无法再通过该师父继续学习，转入自修：

| 途径 | 方式 |
|------|------|
| 读书 | 研读对应武学书籍 / 心得，消耗道具 |
| 实战顿悟 | 在实战中按一定条件 / 概率领悟提升 |

- 自修**仅在师父可教上限封顶后开放**，定位为后期长线成长，**不替代**师父的前中期教学。
- 自修最高仍受**秘籍解锁上限**封顶。

## 8. 武学与武器、四维

- **武器绑定**：每门武学绑定一种武器类型（拳 / 棍 / 剑 / 鞭 / 暗器 / 刀…），需装备对应武器方可施展。
- **四维驱动**：武学效果通过角色四维结算——力道→外功攻防、气劲→内功攻防与内力、根骨→气血与生存、身法→命中闪避会心招架（详见《角色系统》）。
- **本门优先**：默认玩家**只能修习当前所属门派的武学**；跨门派 / 通用武学（如轻功、心法类）是否开放，作为后续扩展，**待定**。

## 9. 战斗使用（接口，归《战斗系统》）

以下规则本文只标接口，具体在《战斗系统》定义：

- 战斗中可携带 / 装备的武学数量（技能栏位）。
- 主动武学的**内力消耗、冷却、前后摇、连招衔接**。
- 被动武学（内功 / 护体）的常驻生效方式。
- 武学间的**增益叠加、互斥、克制**规则。
